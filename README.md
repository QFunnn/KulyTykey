# Fourier Audio Analyzer

Экспериментальный проект по анализу аудиосигналов на основе преобразования Фурье.

Основная цель проекта — представить звук не как непрерывный поток данных, а как набор частотных компонентов, которые могут быть сохранены, переданы или обработаны в более компактной форме. 
Такой подход открывает возможности для реализации альтернативных способов передачи голоса и аудиоданных в игровых средах без необходимости модификации клиентской части.
## Возможности

* Загрузка и обработка WAV-аудио.
* Разложение сигнала на частотные компоненты.
* Покадровый спектральный анализ.
* Поддержка оконных функций для повышения качества анализа.
* Формирование структурированных результатов для дальнейшего использования.
